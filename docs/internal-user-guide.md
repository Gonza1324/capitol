# Capitol Hub - Guia breve de uso interno

Capitol Hub es la plataforma interna de Capitol para ordenar clientes, contactos, tareas, interacciones, reportes, alertas, stakeholders, documentos e historial de trabajo.

La app sirve para centralizar informacion validada y consultable por el equipo. Conviene cargar datos operativos, responsables, proximos pasos, documentos relevantes y el registro de lo trabajado con cada cliente.

## Flujo recomendado

1. Crear el cliente con datos generales, estado, tipo, rubros, issues y responsables internos.
2. Cargar contactos principales y secundarios.
3. Crear tareas activas con responsable, prioridad y fecha limite.
4. Registrar calls/interacciones relevantes con resumen, decisiones, riesgos y proximos pasos.
5. Crear tareas derivadas cuando una interaccion, reporte o alerta deje una accion concreta.
6. Registrar reportes enviados con cliente, destinatarios, responsable, fecha y link/documento.
7. Registrar alertas enviadas con urgencia, canal, clientes, rubros, issues y destinatarios.
8. Vincular stakeholders relevantes al cliente.
9. Subir documentos o guardar links externos con nombres claros.
10. Consultar el historial del cliente para entender que se trabajo.
11. Usar busqueda global para encontrar clientes, contactos, tareas, interacciones, reportes, alertas, stakeholders y documentos.

## Buenas practicas

- Revisar si el cliente, contacto o stakeholder ya existe antes de crear uno nuevo.
- Usar nombres consistentes y faciles de buscar.
- Cargar resumen claro despues de cada call o reunion importante.
- Registrar proximos pasos como tareas, no dejarlos solo en notas.
- Asociar reportes y alertas al cliente correcto.
- Nombrar documentos y links con contexto suficiente.
- Archivar registros que ya no esten activos en vez de duplicarlos o borrarlos.
- No cargar informacion fuera del cliente o entidad correspondiente.
- Mantener observaciones sensibles breves, relevantes y necesarias.

## Fuera de V1

En esta version no estan activos:

- Honorarios.
- Portal de clientes.
- Google Calendar.
- Gmail.
- Google Drive API.
- WhatsApp.
- IA.

Reportes, alertas, documentos y links se registran manualmente.
